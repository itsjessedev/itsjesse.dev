POST /api/v1/classify

Classify a single text input and return category predictions with confidence scores.

Parameters

Parameter Type Required Description
text string Required The text content to classify
include_sentiment boolean Optional Include sentiment analysis (default: false)
include_keywords boolean Optional Extract keywords (default: false)
threshold float Optional Minimum confidence threshold 0-1 (default: 0.5)

Request Example

cURL 
curl -X POST https://api.smartclassify.io/v1/classify \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "I love this product!",
    "include_sentiment": true,
    "include_keywords": true
  }'
Python 
import requests

url = "https://api.smartclassify.io/v1/classify"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "text": "I love this product!",
    "include_sentiment": True,
    "include_keywords": True
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(result)

POST /api/v1/batch

Submit a batch of texts for classification. Returns a job ID for tracking progress.

Request Example

JavaScript 
const response = await fetch('https://api.smartclassify.io/v1/batch', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    texts: [
      "Great customer service!",
      "App keeps crashing",
      "How much does it cost?"
    ]
  })
});

const data = await response.json();
console.log(data.job_id);
