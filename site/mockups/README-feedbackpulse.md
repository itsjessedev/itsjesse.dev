# FeedbackPulse Mockups

Professional HTML mockups for the FeedbackPulse project - a Review & Survey Analyzer.

## Project Description

**Problem:** Customer feedback scattered across Google Reviews, Yelp, surveys, and support tickets with no unified view

**Solution:** Aggregate all feedback sources, run sentiment analysis, extract keywords, and alert on negative trends

**Tech Stack:** Python, React, OpenAI, Google Places API, Yelp API

## Mockup Files

All mockups use TailwindCSS from CDN and Inter font, with a consistent pink/rose theme.

### HTML Files (in `/mockups/`)

1. **feedbackpulse-dashboard.html** - Overview with sentiment score, recent feedback, trend charts
2. **feedbackpulse-reviews.html** - All reviews aggregated from Google, Yelp with sentiment tags
3. **feedbackpulse-surveys.html** - Survey responses with sentiment analysis
4. **feedbackpulse-trends.html** - Trend analysis with keyword clouds, sentiment over time
5. **feedbackpulse-alerts.html** - Negative feedback alerts, AI-powered response suggestions
6. **feedbackpulse-sources.html** - Connected sources (Google, Yelp, surveys) configuration

## Features Showcased

- **Sentiment Analysis:** Real-time sentiment scoring (0-10 scale)
- **Multi-source Aggregation:** Google Reviews, Yelp, Custom Surveys, Support Tickets
- **Trend Tracking:** Charts showing sentiment over time
- **Keyword Extraction:** Automatically identified topics and themes
- **AI-Powered Alerts:** Critical alerts with suggested responses
- **Visual Keyword Clouds:** Size-based frequency visualization
- **Source Management:** Integration configuration and sync controls

## Design Elements

- **Color Scheme:** Rose/pink gradient sidebar, clean white content area
- **Typography:** Inter font family throughout
- **Navigation:** Consistent sidebar with active state indicators
- **Data Visualization:** Chart.js for interactive charts
- **Status Indicators:** Color-coded sentiment badges (green/yellow/red)
- **Response UI:** Progress bars, badges, cards with realistic data
