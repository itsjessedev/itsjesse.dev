# StockAlert Mockups

Created: December 26, 2024

## Overview
Six professional HTML mockups for the StockAlert inventory monitoring system project.

## Project Description
**StockAlert** - Inventory Monitoring System
- **Problem:** Retail chain experienced frequent stockouts and overordering due to manual inventory tracking
- **Solution:** Connected POS systems across locations, monitor levels in real-time, alert on low stock, suggest reorder quantities based on velocity
- **Tech:** Python, Square API, Twilio, Slack Webhooks

## Mockup Files

### HTML Mockups (in `/mockups/`)
1. `stockalert-dashboard.html` - Overview with inventory levels, alerts, reorder suggestions
2. `stockalert-inventory.html` - Full inventory list with stock levels, velocity, last sold
3. `stockalert-locations.html` - Multi-location view showing stock by store
4. `stockalert-alerts.html` - Active alerts, alert history, threshold configuration
5. `stockalert-reorder.html` - Smart reorder suggestions with quantities based on sales velocity
6. `stockalert-settings.html` - Square connection, Twilio/Slack notifications, threshold settings

## Design Features
- **Framework:** TailwindCSS (via CDN)
- **Font:** Inter from Google Fonts
- **Theme:** Amber/Orange gradient sidebar
- **Layout:** Consistent sidebar navigation across all pages
- **Data:** Realistic retail inventory data (organic milk, coffee beans, yogurt, etc.)
- **Features:**
  - Real-time stock level indicators
  - Velocity-based reorder calculations
  - Multi-location inventory comparison
  - Alert threshold configuration
  - Integration settings (Square, Twilio, Slack)

## Usage
Open any HTML file directly in a browser to view the mockup. All styling is self-contained with TailwindCSS CDN.
